Struct bigtwo::trick::Trick

struct Trick {
    hand: Vec<Hand>,
    current_player_id: usize,
    passed_player_ids: BTreeSet<usize>,
}
Expand description

Represents the current state of a Trick, keeps track of which hands have been played and who has passed, and who is the current player.

Fields

hand: Vec<Hand>

The hand used to start the Trick, and all following played hands.

current_player_id: usize

Used to index into a [Player; NUM_PLAYERS] which is passed into functions TODO: (maybe) use lifetimes and a reference to [Player; NUM_PLAYERS].

passed_player_ids: BTreeSet<usize>

Keeps track of all players who have passed so far this Trick

fn do_trick(&mut self, players: &mut [Player; 4]) -> TrickResult

Used to perform the entirety of the Trick, running all Player’s turns, collecting their Hands, keeping track of their Passes, and ending when the Game ends, or when all but one Player has passed.

source

fn do_player_turn(&mut self, players: &mut [Player; 4])

Used to collect a Player’s Hand (or Pass) on their turn.

Panics
  • If there are fewer than 2 players remaining in the Trick (i.e. have not passed)
  • If any of the players have 0 cards (this would mean the game is already over)/
source

fn is_trick_over(&self, players: &[Player; 4]) -> StepStatus

Returns StepStatus::GameOver if a player has 0 cards (that player has won). Returns StepStatus::TrickOver if only one player remains in the trick (3 have passed).

source

fn get_submitted_hand(player: &Player, hand_to_beat: &Hand) -> Hand

Keep asking the player to submit a Hand, until they submit a legal playable Hand. (i.e. keep getting their inputs until they submit valid input that can progess the game).
